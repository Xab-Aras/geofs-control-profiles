# GeoFS Control Profiles

Save and switch multiple GeoFS control/joystick profiles using Tampermonkey.

## Install
1. Install Tampermonkey (Chrome/Firefox/Edge)
2. Open this link:
   - https://raw.githubusercontent.com/Xab-Aras/geofs-control-profiles/main/geofs-control-profiles.user.js
3. Confirm install in Tampermonkey

## Usage
- Type a profile name and click **Save**
- Select a profile and click **Load & Reload**
- **Export JSON** copies the profile settings to clipboard

## Notes
Profiles are stored in browser `localStorage` under keys starting with `gcp_profile_`.
